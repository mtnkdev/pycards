assets.py¶

This module is responsible for loading and parsing the cardset configurations available to the program

cardsets must be located in the ‘cardsets’ directory in the root of the source directory

source.model.assets.load_cardsets()¶

Load and parse the various cardset configurations Store the cardsets but avoid loading the images until the cardset is in use for a game
